# Страница погоды для портфолио

Красивая страница погоды на React: динамический фон и анимации в зависимости от времени суток и погоды, график температуры на неделю (Recharts).

## Возможности

- **Погода по геолокации** — используется бесплатный API [Open-Meteo](https://open-meteo.com/) (без ключа). Если геолокация недоступна, показывается погода для Москвы.
- **Фон и анимации**:
  - **День + ясно** — голубое небо, солнце с лёгкой пульсацией.
  - **День + облачно** — облака, солнце.
  - **День + дождь/гроза** — серый фон, анимация дождя; при грозе — вспышки молнии.
  - **День + снег** — холодные тона, анимация снежинок.
  - **Ночь + ясно** — тёмный градиент, луна, звёзды с мерцанием.
  - **Ночь + дождь/гроза** — тёмный фон, дождь, молния.
- **График на неделю** — две линии (макс/мин температура) с заливкой, тултип по дням. Стили графика подстраиваются под день/ночь.

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно `http://localhost:5173`).

## Сборка

```bash
npm run build
npm run preview
```

## Стек

- React 18, Vite
- Recharts — график температуры
- Open-Meteo API — данные погоды
- CSS — анимации (дождь, снег, облака, солнце, луна, звёзды, молния)
